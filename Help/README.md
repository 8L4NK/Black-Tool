### Installing Black-Tool
``` sh
git clone https://github.com/mrprogrammer2938/Black-Tool

cd Black-Tool

python3 install.py
```
<br>

### Run Black-Tool
``` sh
hack --start
```

#### Black-Tool Argument:
<pre>
hack --help

Black-Tool Usage:
                  --start   | start Hack Menu!
                  --install   <Pkg Name>
                  --clone     <Repositories Link>
                  --version | Black-Tool version
                  --help    | Black-Tool Help
                  --banner  | Show Black-Tool Banner
                  --scan portscan | dos
</pre><br>
